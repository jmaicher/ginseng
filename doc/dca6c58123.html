addModule

Adds a new module instance from the given module constructor with the given options to the instance. If the autoload parameter is true the module gets automatically loaded. When the paramter is set to false, it must be loaded manually. If autoload is a String, it is expected to be a name of an existing route from #router. Then, the module will automatically be loaded when #router triggers matching route events.
Parameters:
